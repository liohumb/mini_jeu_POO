Très simple !  
Sur ton terminal, tu peux lancer :  

Le premier programme :  
`ruby app.rb`  

Le deuxième programme :  
`ruby app_2.rb`

Le troisième et dernier programme :  
`ruby app_3.rb`

Tu veux quitter avant la fin ?  
Pas de soucis ! Il te suffit de faire CTRL + C
